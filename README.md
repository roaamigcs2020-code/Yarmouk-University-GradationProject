# Graduation Project: Sophia (Intelligent Voice Assistant)

Sophia is an interactive, intelligent desktop voice assistant designed to help users seamlessly open applications, navigate websites, and execute Google searches using either voice commands or text input. The project aims to bridge the gap between frontend interactivity and backend processing while managing structured automation data.

## 🚀 Key Features & Functionality
* **Speech Processing & NLP:** Translates user voice commands into system actions or text queries.
* **Dual Input System:** Supports both microphone voice activation and manual text-based queries.
* **Database-Driven Execution:** Utilizes a central database (`sophia.db`) to dynamically map user commands to the correct system paths, applications, or web URLs.
* **User-Friendly Desktop Interface:** Features a clean, accessible layout focusing on speed and intuitive user interaction.

## 🛠️ Tech Stack & Architecture
* **Backend & Logic:** Python (handles core speech processing, command parsing, and system automation)
* **Frontend & Interactivity:** JavaScript (manages responsive client-side UI elements)
* **Database Management:** Relational Database (stores file paths, application mappings, and pre-configured website URLs)
